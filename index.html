Github in a Graph, with Neo4j

Workshop Guide

A Neo4j Workshop using Github as a Data Dource.

As a giant developer playground for collaborative coding, Github represents a few fascinating data sets: a social graph of people and their skillz, a project graph of branches and forks, and a dependency graph of projects.

We'll explore this coding landscape by diving into the data using Neo4j.

This workshop is for graphistas of all skill levels. Participate in a few hours of coding led by experienced Neo4j developers. Learn while you play with github project data, focusing on data patterns, data queries, and visualization.

Goals:

  • learn about Neo4j and graph databases
  • how to import and structure huge data sets
  • how to query data
  • how to visualize data
  • create original graph-based applications
